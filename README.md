匹配模式：^https:\/\/minasigo-no-shigoto-pd-c-res\.orphans-order\.com\/(\d+(?:\.\d+)*\/[^.]+\/[^.]*\.txt)$

匹配类型：正则表达式

7.如果不用了就把这个黑色窗口关闭，并且把拓展关了即可。
